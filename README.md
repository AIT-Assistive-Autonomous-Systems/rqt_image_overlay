# Run

Run rqt with the following:
```
rqt --force-discover
```

From the menubar, add the plugin by navigating through Plugins>Debugging>Image Overlay.

Alternatively, run rqt with the following argument:

```
rqt --force-discover -s ImageOverlay
```
